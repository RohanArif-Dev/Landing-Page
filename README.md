## Prerequisites

Node.js >= 12.x
npm >= 6.x
## Installation

Clone the repository

```bash
  git clone https://github.com/RohanArif-Dev/Landing-Page.git
```
    
Install the dependencies

```bash
  cd Landing-Page
  npm install
```

Run the development server

```bash
  npm run dev
```
he landing page will be available at http://localhost:3000
